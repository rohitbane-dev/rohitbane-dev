* **Creational Logic (Factory Pattern):** Centralized instantiation to decouple the API from specific cache implementations, ensuring clean object creation.
* **Algorithmic Flexibility (Strategy Pattern):** Integrated hot-swappable eviction policies (LRU/LFU) allowing the system to adapt to different data access patterns at runtime.
* **Standardized Execution (Template Pattern):** Defined a rigid execution skeleton for cache operations (Put/Get/Evict) to ensure consistent synchronization and logging across all implementations.
* **Concurrency & Throughput:** Leveraged `ConcurrentHashMap` for high-parallelism and implemented Asynchronous Write-Behind policies to minimize latency.
